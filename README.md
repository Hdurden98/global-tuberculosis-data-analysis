# Global Tuberculosis SQL Case Study

## Project Overview
This project analyzes global tuberculosis data using SQL to identify high-burden countries, regional mortality trends, detection performance, and changes over time.

## Analysis Performed
- Regional case and death comparisons
- Mortality rate calculations
- Country rankings using window functions
- Year-over-year trend analysis
- Population-adjusted incidence analysis
- Detection vs mortality risk analysis

## Skills Demonstrated
- CTEs
- Window functions
- Ranking functions
- Aggregation and filtering
- Public health trend analysis
- Business and policy-focused insights

- `tuberculosis_analysis.sql`
